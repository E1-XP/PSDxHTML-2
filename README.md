## Installation

How to run project locally:

1. Clone repository:
```bash
  git clone git@github.com:E1-XP/PSDxHTML-2.git
```
2. Move inside project directory:
```bash
  cd PSDxHTML-2
```
3. Install dependencies:
```bash
  npm install
```
4. Run development server:
```bash
  npm start
```
